## 문제 풀이
- 입력으로 정점의 개수와 그래프의 인접 행렬이 주어진다.
- i번째에서 j로 가는 경로가 존재할 경우 1, 존재하지 않을 경우 0을 출력

### 1.플로이드 알고리즘
#### i에서 j번째로 가는 최소 경로를 계산한다.
#### 점화식 : d[i][j] = min(d[i][j], d[i][k] + d[k][j])
#### 계산한 결과값이 INF값과 같다면 0출력 그렇지 않다면 1을 출력
