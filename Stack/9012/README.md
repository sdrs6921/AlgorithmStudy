## 문제 풀이
- 한 쌍의 '('와 ')'로 이루어 진 문자열을 기본 VPS
- '(' "VPS" ')' => VPS, VPSVPS => VPS
- 주어진 문자열이 VPS인지 아닌 지 판별

### 문제 해결
#### 1. 먼저 '(' 괄호가 들어 올 경우 Stack에 넣는다.
#### 2. ')'가 들어왔을 떄 만약 스택이 비어있다면 VPS가 아니고, 그렇지 않다면 Pop한다.
#### 3. 마지막으로 스택이 비어있다면 VPS이고, 그렇지 않다면 VPS가 아니다
