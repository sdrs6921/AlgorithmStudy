## 문제 풀이
- 입력 값으로 알파벳 순으로 부모, 왼쪽 자식, 오른쪽 자식이 주어진다.
- 전체 이진 트리가 주어질 때, 전위순회, 중위순회, 후위순회의 결과를 출력한다.

### 배열 자료구조를 사용하여 트리 구조를 표현
#### 1. 왼쪽 자식과 오른쪽 자식을 기록할 배열을 만든다.
#### 2. 재귀 함수를 통해 전위순회, 중위순회, 후위순회를 할 함수를 현하여 main 함수에서 호출한다.
