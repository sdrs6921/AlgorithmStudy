## 문제 풀이
- 문자열 2개가 입력 받음
- 첫 번째 문자열을 무작위로 배치하여 두 번쨰 문자열을 만들 수 있는지 판별

### 무작위로 배치해서 만드려면 모든 알파벳의 개수가 같아야함
#### 1. 첫 번째 문자열의 알파벳 개수를 counting
#### 2. 두 번쨰 문자열을 확인하며 counting한 배열에서 하나 씩 줄임
#### 3. 만약 배열의 개수가 0이 아니라면 불가능 하다고 판별
