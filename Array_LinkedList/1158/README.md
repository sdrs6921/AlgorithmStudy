# Queue_요세푸스 문제

## 문제 풀이
- 입력 값으로 N, K가 주어진다
- N개의 숫자가 원으로 이루어 져 있을 때 K번째 숫자를 없애면서 돌아간다.
- 예를 들어 N 이 4고 k가 3 일때
- 1, 2, 3, 4 에서 3을 없애고
- 4 1 2 에서 2를 없애고
- 4 1 에서 4를 없애고
- 1 에서 1을 없애면서 출력

### 해결 방법
- Queue 자료구조를 활용한다
- K 보다 하나 적은 만큼 poll에서 뒤에 offer한다.
- 그리고 Queue를 poll하면 해당 번째 숫자만큼 원으로 돌아가 면서 뽑을 수 있다
