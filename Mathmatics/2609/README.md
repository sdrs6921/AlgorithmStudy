# Mathmatics_최대공약수, 최소공배수

## 문제 풀이
- 입력 : 두 개의 수
- 출력 : 두 수의 최대 공약수와 최소 공배수를 출력

### 알고리즘
- 최대 공약수 : 유클리드 호제법을 재귀 함수로 구현
- 최소 공배수 : 두 수중에서 큰 수의 배수를 증가시켜 작은 수로 나누어 떨어질때까지 반복문을 돌려 최소 공배수를 구함
